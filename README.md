## 🚀 Quick Start

### Option 1: Open in Browser
1. Open `Animated.html` directly in your favorite web browser
2. Enjoy the animations and explore the portfolio!

### Option 2: Local Setup
```bash
# Navigate to the folder
cd /Downloads/Porfolio2

# Open with a local server (optional for best results)
# Using Python 3
python3 -m http.server 8000

# Then visit: http://localhost:8000/Animated.html
```

### ⚙️ Prerequisites
- Ensure `MyIntro.mp4` is in the same folder as `Animated.html`
- Modern web browser with JavaScript enabled
- Video file properly encoded for web playback

## 📸 Screenshots

You can take screenshots of this portfolio by:

### 📱 Browser Screenshot Methods

**Chrome / Edge / Safari / Firefox:**
1. **Full Page**: `Cmd + Shift + S` (Mac) or `Ctrl + Shift + S` (Windows)
2. **Active Window**: `Cmd + Shift + 3` (Mac) or use Snip & Sketch (Windows)
3. **Selected Area**: `Cmd + Shift + 4` (Mac) and drag to select

**Using Developer Tools:**
1. Press `F12` or `Cmd + Option + I` to open DevTools
2. Use the screenshot tool (camera icon in DevTools)
3. Choose "Capture full page screenshot"

### 🎥 Recommended Screenshots to Capture
- **Hero Section** with video background and animations
- **Skills Section** highlighting your expertise
- **Project Cards** in their hover state
- **Full Page** for social media sharing
- **Mobile View** (toggle device toolbar in DevTools)

## 🎨 Customization Guide

### 📝 Text Changes
- Update name, title, and bio in the HTML

### 🔗 Social Links & Contact
Replace placeholder links with your real URLs:
- **LinkedIn** → Your professional profile
- **GitHub** → Your code repository
- **DevBuddy** → Your content creation platform
- **Email** → Your contact email

### 🎭 Visual Customization
- **Color Scheme**: Modify the CSS color variables for brand consistency
- **Fonts**: Change typography to match your personal style
- **Animations**: Adjust animation duration and timing functions

### 📸 Video Replacement
- Replace `MyIntro.mp4` with your own intro video
