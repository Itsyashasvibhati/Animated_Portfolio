# 🌟 Animated Portfolio

Welcome to the 🤩 cute and colorful portfolio page built in `doodle2.html`!

This project is a playful digital showcase for **Yashasvi Bhati** — a QA Engineer, Content Creator, and founder of **BNB Global**.

## ✨ What’s inside

- A smooth hero section with a **video background** using `MyIntro.mp4`
- A custom **hand-drawn doodle style** with stickers, floating cards, and fun animation
- A delightful **cursor effect** with a dot + ring that follows the mouse
- A quirky introduction section with achievements and mission statement
- A creative skills section featuring QA, programming, and tools
- Project cards, content creator section, and contact links

## 💖 Why this is special

This page is made to feel friendly, energetic, and memorable — just like a handwritten portfolio with cheerful doodles.

## 🚀 How to use

1. Open `doodle2.html` in your browser.
2. Make sure `MyIntro.mp4` is in the same folder so the hero video loads correctly.
3. Change the email, links, or text anytime to personalize the page.

## 🛠️ Files

- `doodle2.html` — main portfolio page
- `MyIntro.mp4` — hero background video (should be in the same folder)

## 💡 Tips for customization

- Replace the sample social links with your real **LinkedIn**, **GitHub**, and **DevBuddy** URLs.
- Tweak the colors in the CSS to match your brand.
- Add real project links or case studies to the project cards.

Thanks for exploring this cheerful portfolio page! 🌈
